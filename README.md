# AURUM by BYD

Apresentação digital interativa da proposta de sociedade entre AURUM Cinema e BYD. O projeto combina narrativa comercial, sistema oficial de co-branding, visualizações conceituais do cinema e motion design orientado por scroll.

## Experiência publicada

https://aurum-by-byd-investment.rubensafnrv.chatgpt.site

## Stack

- React 19 + TypeScript
- Vinext / Vite
- GSAP + ScrollTrigger
- CSS responsivo com fontes AURUM auto-hospedadas

## Desenvolvimento

```bash
npm install
npm run dev
```

## Validação

```bash
npm run lint
npm test
```

## Identidade

As assinaturas aprovadas estão em `public/brand/official`. A apresentação usa a versão horizontal em navegação e rodapé, a versão unificada OP1 como assinatura principal e as famílias AURUM Cinema e AU + BYD em aplicações institucionais e compactas.

As imagens arquitetônicas são visualizações conceituais e não representam projeto executivo.
